# Developer Skills Survey Form

A responsive HTML/CSS survey form built as part of my 6‑month backend development journey.

##  Live Demo
https://jocular-sfogliatella-2b0813.netlify.a

##  Tech Stack
- HTML5 (semantic form tags)
- CSS3 (Flexbox, responsive design)
- Deployed on Netlify

##  Features
- 8 different input types: text, email, number, dropdown, radio buttons, checkboxes, textarea, submit button
- HTML5 form validation (required fields, email format, age range)
- Fully responsive – works on mobile, tablet, and desktop
- Accessible labels and semantic structure

##  What I Learned
- Structuring a real‑world form with proper `<label>` associations
- Using different input types and their validation attributes
- Organizing CSS with a mobile‑first approach
- Deploying a static site manually via Netlify
